# First-party Plugins

This directory contains source code for first-party R3TRIVE plugins.

See [PLUGIN_SDK.md](../PLUGIN_SDK.md) for plugin development documentation.

## Planned Plugins

| Plugin | Type | Description |
|---|---|---|
| splunk | output | Forward events to Splunk |
| elastic | output | Forward events to Elasticsearch |
| jira | action | Create Jira tickets from incidents |
| pagerduty | action | Alert via PagerDuty |
| virustotal | intelligence | VirusTotal hash lookup |
| misp | intelligence | MISP threat intelligence feed |
